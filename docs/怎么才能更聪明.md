1. 吸收

   大量阅读，广泛获取信息

2. 内化

   将获取的知识、信息，通过自己的思考，内心演练进行过滤、内化

3. 输出

   将内化后的信息进行文字呈现，向他人讲清楚

4. 归类

   将不同的收获进行分门别类

5. 改进

   在实际运用中不断进化

6. 总结

   将改进后的东西再次总结，让它彻底成为自己的东西
